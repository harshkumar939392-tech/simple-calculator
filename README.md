# 🧮 Simple Calculator

A simple command-line calculator built using Python. This project performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- User-friendly input
- Beginner-friendly Python project

## 🛠️ Technologies Used

- Python 3

## 📂 Project Structure

```
simple-calculator/
│── calculator.py
│── README.md
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/harshkumar939392-tech/simple-calculator.git
```

2. Open the project folder:

```bash
cd simple-calculator
```

3. Run the program:

```bash
python calculator.py
```
